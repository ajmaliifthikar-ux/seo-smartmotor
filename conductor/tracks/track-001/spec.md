# Specification: Phase 1: Foundation

## Goal
A working luxury website with real booking capability.

## Deliverables
1.  **Project Setup:** Next.js 15 (App Router), TypeScript, Prisma, PostgreSQL, Firebase Auth, Tailwind CSS, Shadcn UI.
2.  **Public Marketing Pages:** Home, About, Services, Contact (using the design system).
3.  **Booking Engine:** 4-screen flow (Choose Service, Vehicle, Time, Confirm) with real-time availability.
4.  **Admin Shell:** Sidebar navigation, Dashboard placeholder, Basecamp for booking management.
5.  **Deployment:** Vercel CI/CD pipeline setup.

## Exit Criteria
- A customer can discover Smart Motor, browse services, and book an appointment.
- An admin can view and manage bookings.
